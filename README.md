# 🎓 Student Management System - ASP.NET Core Web API

## 📌 Project Overview
This project is a Student Management System built using ASP.NET Core Web API.  
It allows managing student records using CRUD operations with a clean layered architecture.

---

## 🚀 Features

- Create a new student
- Retrieve all students
- Retrieve student by ID
- Update student details
- Delete student record
- Global Exception Handling using Middleware
- Swagger API documentation for testing APIs
- Layered Architecture (Controller, Service, Repository)
- SQL Server database integration

---

## 🏗️ Architecture

The project follows a **Layered Architecture**:

- **Controllers** → Handle HTTP requests
- **Services** → Business logic layer
- **Repositories** → Database operations
- **Models** → Data structure
- **Data Layer** → DbContext configuration
- **Middleware** → Exception handling

---

## 🛠️ Technologies Used

- ASP.NET Core Web API (.NET 10.0)
- C#
- Entity Framework Core
- SQL Server
- Swagger (Swashbuckle)
- Git & GitHub
